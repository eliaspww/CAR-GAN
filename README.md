## Cascade Attention Guided Residue GAN for Cross-Modal Translation
### Requirements
- pytorch 0.4+
- python 3.6.8
- visdom 0.1.8.8
- pillow 5.4.1

### Train
`$ sh ltrain.sh`
### Test
